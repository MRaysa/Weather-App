Project Title :

Live Weather Visualization Dashboard

Discription :

This project is an interactive web application that visualizes current weather conditions across major cities in Asia. It integrates the OpenWeatherMap API with Mapbox to display a dynamic map where each city is marked with a colored circle based on its weather condition. Users can hover over or click on markers to view temperature and weather details, and refresh the data in real time. The application is fully responsive and includes features such as a legend, zoom-level scaling, and error handling for API issues.
