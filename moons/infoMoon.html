Project Title:

Live Weather Visualization Dashboard

Description:

This project is an interactive web application that visualizes current weather conditions across major cities in Asia. It integrates the OpenWeatherMap API with Mapbox to display a dynamic map where each city is marked with a colored circle based on its weather condition. Users can hover over or click markers to view temperature, cloud coverage, and other weather details, with real-time data refresh. The application is fully responsive and includes features such as a legend, zoom scaling, and error handling.
