Detailed breakdown of weather project functionality and scores

🔧 Project Features

  • Real-time weather data visualization
  • Interactive Mapbox limited to Asian countries
  • Hover-based city weather data preview
  • Live overlays for temperature, clouds, etc.
  • City-level detailed weather forecast
  • D3.js-based historical data visualization
  • Fully responsive and mobile-friendly design
  • Loading spinner while fetching data
  • Weather markers color-coded by condition
