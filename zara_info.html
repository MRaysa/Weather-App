Weather Visualization Dashboard

An interactive dashboard that displays real-time weather data with beautiful visualizations. The application fetches live weather data from OpenWeatherAPI, displays it on a Mapbox map, and creates dynamic charts using D3.js.

GitHub Repository

https://github.com/MRaysa/Weather-App

Technologies Used

HTML CSS JavaScript Mapbox OpenWeatherAPI D3.js

Project Features

  • Real-time weather data visualization
  • Interactive map with weather overlays
  • Historical weather data charts
  • Responsive design for all devices
  • Customizable dashboard interface
